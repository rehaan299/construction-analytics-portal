# Training Guide

## For Field Users (Foremen/Superintendents)
1. Login
2. Go to Projects
3. Choose your project
4. Open “Daily Report”
5. Enter:
   - Labor Hours
   - Equipment Hours
   - Progress %
   - Notes (issues, delays, safety flags)
6. Submit
7. Confirm submission

Tips:
- Keep notes concise and operational (weather delay, delivery issues, equipment downtime)
- Enter reports daily at end-of-shift

## For Project Managers
1. Login
2. Go to Dashboard
3. Select project
4. Review:
   - Budget used %
   - Latest progress
   - Alerts
5. If alert appears, coordinate with field + finance

## For Finance / Cost Control
1. Login
2. Dashboard → monitor budget burn across projects
3. Review cost spikes and verify categorization
